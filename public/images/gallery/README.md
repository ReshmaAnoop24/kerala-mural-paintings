# Gallery Images Directory

This directory is for storing Kerala mural painting images.

## Image Structure

Place your gallery images here with the following naming convention:
- `mural-1.jpg`, `mural-2.jpg`, etc. for full-size images
- `mural-1-thumb.jpg`, `mural-2-thumb.jpg`, etc. for thumbnails

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Image Recommendations

- Full images: 1200x1200px or larger, optimized for web
- Thumbnails: 400x400px, optimized for web
- Maintain aspect ratio (square format recommended)

## Current Placeholder Structure

The gallery currently uses placeholder graphics. Replace these with actual images by:
1. Adding images to this directory following the naming convention
2. Updating the image paths in `data/gallery.ts` if needed
